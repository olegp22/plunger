#🚀 Plunger Dash

Plunger Dash — это динамичный 2D-платформер, разработанный на C# (.NET) с использованием Windows Forms и кастомной физики.
Главная особенность игры — уникальная механика перемещения с помощью присоски (вантуза), позволяющая игроку разгоняться, цепляться за поверхности и буквально пролетать через уровень на огромной скорости.

#🎮 Основная идея

Игрок управляет персонажем по имени Digger — исследователем в ярком костюме с банановой стилистикой.

Вместо стандартных прыжков игра использует систему Grappling Hook / Plunger Movement, где основой геймплея становятся:

инерция
скорость
точность зацепов
вертикальное перемещение

Главная цель — максимально быстро проходить уровни, собирая монеты и используя окружение для ускорения.

#⚡ Особенности геймплея
#🪠 Система присоски (Grappling Hook)

Игрок может цепляться:

за пол
за потолок
за стены
за платформы
за любые твёрдые поверхности уровня

После зацепа персонаж:

подтягивается к точке
сохраняет импульс
может выполнять быстрые перелёты между объектами
#🌀 Динамика и инерция

Геймплей построен вокруг:

скорости
плавного движения
сохранения импульса
непрерывного перемещения

Чем точнее игрок использует присоску:

тем быстрее проходит уровень
тем эффективнее использует маршруты
тем выше итоговый результат
#🧗 Вертикальный платформинг

Уровни проектируются с упором на:

вертикальное перемещение
использование стен и потолков
сложные маршруты
быстрые смены высоты

Игрок постоянно перемещается между:

нижними платформами
центральными маршрутами
верхними секциями уровня

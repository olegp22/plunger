# Plunger Dash
Plunger Dash — это динамичный 2D-раннер, разработанный на языке C# с использованием графической библиотеки Windows Forms. Проект реализован в рамках учебного курса по объектно-ориентированному программированию и архитектурным паттернам.

# 📝Описание игры
Игрок управляет персонажем, который автоматически бежит вперед с постоянной скоростью. Основная механика заключается в использовании специальной присоски (вантуза). С её помощью игрок может цепляться за объекты игрового мира, чтобы преодолевать пропасти, изменять траекторию движения и собирать бонусы.

# Ключевые особенности:
Автоматический бег: Скорость задана системой, что требует от игрока быстрой реакции.

Механика присоски: Уникальное взаимодействие с окружением на основе физики маятника.

Сбор монет: На каждом уровне разбросаны монетки, которые увеличивают итоговый счет игрока.

Архитектура MVP: Четкое разделение бизнес-логики, данных и интерфейса.

🛠 Технологический стек
Язык программирования: C# (версия .NET 6+)

Фреймворк: Windows Forms (WinForms)
